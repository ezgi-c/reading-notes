# Introduction to SQL

SQL stands for Structured Query Language. It is a language used to query, manipulate and transform data from a relational database. A relational database is a collection of two-dimensional tables of related data. A table consists of rows and columns, rows representing each instance of an entity while columns represent properties shared by all instances of the entity.
